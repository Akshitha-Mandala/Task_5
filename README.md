# 🚢 Titanic Dataset - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to uncover patterns, relationships, and key factors that influenced passenger survival. Using Python and popular data analysis libraries, the dataset is explored through statistical summaries and visualizations.

The project was completed as part of a **Data Analytics Internship** to strengthen skills in data exploration and visualization.

---

* Understand the structure of the Titanic dataset.
* Perform statistical analysis using descriptive measures.

## 📈 Key Insights

* The dataset contains missing values in the **Age**, **Cabin**, and **Embarked** columns.
* Female passengers had significantly higher survival rates than male passengers.
* First-class passengers had the highest probability of survival.
* Third-class passengers experienced the greatest number of fatalities.
* Higher ticket fares were generally associated with better survival chances.
* Most passengers were between **20 and 40 years** of age.
* The **Fare** feature contains several outliers.
* Southampton was the busiest embarkation port.
* Passenger class, gender, and fare were the strongest factors associated with survival.
